二. 知识讲解

1.1.1. for语句的格式、功能

1.11. 11.1 for语句的格式

for 语句的格式如下：

for ( 表达式 1; 表达式 2; 表达式 3) {
        循环体
}
1.21. 21.2 for语句的含义

它的运行过程为：

(1)(1)(1) 先执行“表达式111”。
(2)(2)(2) 再执行“表达式222”，如果它的值为真（非000），则执行循环体，否则结束循环。
(3)(3)(3) 执行完循环体后再执行“表达式3”。
(4)(4)(4) 重复执行步骤 (2)(2)(2) 和 (3)(3)(3)，直到“表达式222”的值为假，就结束循环。

上面的步骤中，(2)(2)(2) 和 (3)(3)(3) 是一次循环，会重复执行，for 语句的主要作用就是不断执行步骤 (2)(2)(2) 和 (3)(3)(3)。

“表达式111”仅在第一次循环时执行，以后都不会再执行，可以认为这是一个初始化语句。
“表达式222”一般是一个关系表达式，决定了是否还要继续下次循环，称为“循环条件”。
“表达式333”很多情况下是一个带有自增或自减操作的表达式，以使循环条件逐渐变得“不成立”。

  • for循环中可以在初始化语句中定义变量,但在该语句中定义的变量只能在循环体内使用而不能超出循环体使用。

  • 如果重复执行的操作（循环体）次数是固定的、已知的，则一般使用 for 语句。

2.2.2. for语句的应用实例

#include <bits/stdc++.h>
int main() {
    int sum=0;
    for(int i = 1; i <= 100; i++){
        sum += i;
    }
    cout << sum << endl;
    return 0;
}
  • 代码解释
  • 执行到 for 语句时，先给 iii 赋初值111，判断 i≤100i\le100i≤100 是否成立；因为此时 i=1，i≤100i=1，i\le100i=1，i≤100 成立，所以执行循环体。循环体执行结束后（sumsumsum的值为111），再计算 i++。
  • 第二次循环时，iii 的值为 222，i≤100i\le100i≤100 成立，继续执行循环体。循环体执行结束后（sumsumsum的值为333），再计算 i++。
  • 重复执行步骤 (2)(2)(2)，直到第101101101次循环，此时 iii 的值为101101101，i≤100i\le100i≤100 不成立，所以结束循环。
  • 3个表达式可以同时省略,for( ; ; )相当于while(1)

3.3.3. 三种循环语句的综合应用

从本质上来说，三种循环都是可以互相替代的
但我们实际应用中，应该选取最适合我们的循环语句来实现程序

对于 while 和 do-while 循环，循环体中应包括使循环趋于结束的语句。

对于 while 和 do-while 循环，循环变量的初始化操作应该在 while 和 do-while 语句之前完成，而 for 循环可以在内部实现循环变量的初始化。

for 循环是最常用的循环，它的功能强大，一般都可以代替其他循环
